## Admin wachtwoord reset
Op `login.html` zit nu een knop **Wachtwoord vergeten?**.
Werking:
1. Vul het admin e-mailadres in.
2. Klik op **Wachtwoord vergeten?**
3. Klik op **Verstuur resetmail**

Voorwaarde:
- Firebase Authentication met **Email/Password** moet aanstaan.
- Het domein van je site moet in Firebase Authentication bij **Authorized domains** staan.
